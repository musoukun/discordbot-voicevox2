# VoiceVox Bot 使い方ガイド

VOICEVOX音声合成とAI回答機能を備えたDiscordボットです。

## コマンド一覧

### `/vv` - テキスト読み上げ

テキストをVOICEVOXで音声合成し、ボイスチャンネルで読み上げます。

| オプション | 必須 | 説明 |
|---|---|---|
| `text` | はい | 読み上げる文章 |
| `speaker` | いいえ | 話者の選択（デフォルト: ずんだもん (ノーマル)） |
| `custom_speaker` | いいえ | カスタム話者名（`speaker`で「カスタム」を選択した場合） |
| `channelid` | いいえ | ボイスチャンネルのID（省略時は自分が接続中のVC） |
| `speed` | いいえ | 話速 0.5〜2.0（デフォルト: 1.0） |
| `pitch` | いいえ | 音高 -0.15〜0.15（デフォルト: 0） |
| `intonation` | いいえ | 抑揚 0〜2.0（デフォルト: 1.0） |
| `volume` | いいえ | 音量 0〜2.0（デフォルト: 1.0） |

**使用例:**
```
/vv text:こんにちは世界
/vv text:ゆっくりしていってね speaker:四国めたん (ノーマル) speed:0.8
```

---

### `/vvai` - AI質問 + 読み上げ

Gemini AI（gemini-2.5-flash）に質問し、回答をVOICEVOXで読み上げます。
ユーザーごとに会話履歴が保持されるため、文脈を踏まえた連続した会話が可能です。

| オプション | 必須 | 説明 |
|---|---|---|
| `question` | はい | AIへの質問 |
| `speaker` | いいえ | 話者の選択（デフォルト: ずんだもん (ノーマル)） |
| `channelid` | いいえ | ボイスチャンネルのID |
| `search` | いいえ | Google検索を使って最新情報を補完するか（デフォルト: false） |

**使用例:**
```
/vvai question:今日の天気を教えて
/vvai question:最新のニュースは？ search:True
```

---

### `/vvaiq` - ローカルAI質問 + 読み上げ

ローカルで動作するAI（KoboldAI + Qwen3.5-9B）に質問し、回答をVOICEVOXで読み上げます。
インターネット接続不要で、プライベートな質問にも対応できます。

| オプション | 必須 | 説明 |
|---|---|---|
| `question` | はい | AIへの質問 |
| `speaker` | いいえ | 話者の選択（デフォルト: ずんだもん (ノーマル)） |
| `channelid` | いいえ | ボイスチャンネルのID |

**使用例:**
```
/vvaiq question:プログラミングの勉強法を教えて
```

---

### `/lvv` - ボイスチャンネルから退出

ボットをボイスチャンネルから退出させます。
10分間操作がない場合は自動的に退出します。

**使用例:**
```
/lvv
```

---

## 基本的な使い方

1. Discordのボイスチャンネルに参加する
2. テキストチャンネルでスラッシュコマンドを入力する
3. ボットが自動的にボイスチャンネルに参加し、音声を再生する
4. 使い終わったら `/lvv` で退出させるか、10分間放置で自動退出

## 注意事項

- コマンドは1ユーザーにつき1つずつ処理されます。前のコマンドが完了するまで次のコマンドは実行できません。
- ボイスチャンネルに参加していない状態でコマンドを実行する場合は `channelid` オプションでチャンネルIDを指定してください。
- `/vvai` の `search` オプションを有効にすると、Google検索で最新情報を取得してから回答を生成します。
- `/vvaiq` はローカルAIサーバー（KoboldAI）が起動している必要があります。

## 話者について

VOICEVOXには多数の話者が用意されています。コマンドの `speaker` オプションから選択できます。
選択肢にない話者を使いたい場合は、`speaker` で「カスタム」を選び、`custom_speaker` に話者名を正確に入力してください（`/vv` コマンドのみ対応）。

話者名の例:
- ずんだもん (ノーマル)
- 四国めたん (ノーマル)
- 春日部つむぎ (ノーマル)
- 雨晴はう (ノーマル)
- 波音リツ (ノーマル)
