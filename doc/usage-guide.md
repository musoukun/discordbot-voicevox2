# VoiceVox Bot 利用ガイド

VOICEVOX音声合成 + AI回答機能を備えたDiscordボットです。

---

## コマンド一覧

| コマンド | 説明 | AI | 音声 |
|---------|------|:--:|:---:|
| `/vv`   | テキスト読み上げ | - | o |
| `/vvai` | Gemini AIに質問 + 読み上げ | Gemini | o |
| `/vvq`  | Qwen3.5に質問 + 読み上げ | Qwen (ローカル) | o |
| `/q35`  | Qwen3.5に質問（テキストのみ） | Qwen (ローカル) | - |
| `/lvv`  | ボイスチャンネルから退出 | - | - |

全コマンドに `s` を付けると**自分だけに見える**応答になります（例: `/vvs`, `/vvais`, `/vvqs`, `/q35s`）。

---

## `/vv` - テキスト読み上げ

テキストをVOICEVOXで音声合成し、ボイスチャンネルで読み上げます。

| オプション | 必須 | 説明 |
|-----------|:----:|------|
| text | o | 読み上げる文章 |
| speaker | - | 話者の選択（デフォルト: ずんだもん） |
| custom_speaker | - | カスタム話者名（speakerで「カスタム」選択時） |
| channelid | - | ボイスチャンネルのID（省略時は自分のVC） |
| speed | - | 話速 0.5〜2.0（デフォルト: 1.0） |
| pitch | - | 音高 -0.15〜0.15（デフォルト: 0） |
| intonation | - | 抑揚 0〜2.0（デフォルト: 1.0） |
| volume | - | 音量 0〜2.0（デフォルト: 1.0） |

**例:**
```
/vv text:こんにちは世界
/vv text:ゆっくりしていってね speaker:四国めたん (ノーマル) speed:0.8
```

---

## `/vvai` - Gemini AI質問 + 読み上げ

Google Gemini AIに質問し、回答をVOICEVOXで読み上げます。
ユーザーごとに会話履歴が保持され、文脈を踏まえた連続した会話が可能です。

回答が150文字を超える場合、読み上げは**AIが生成した140文字の要約**になります。
Discordには全文と要約の両方が表示されます。

| オプション | 必須 | 説明 |
|-----------|:----:|------|
| question | o | AIへの質問 |
| speaker | - | 話者の選択（デフォルト: ずんだもん） |
| channelid | - | ボイスチャンネルのID |
| search | - | Google検索で最新情報を補完（デフォルト: false） |

**例:**
```
/vvai question:プログラミングの勉強法を教えて
/vvai question:最新のニュースは？ search:True
```

---

## `/vvq` - Qwen3.5 質問 + 読み上げ

ローカルAI（KoboldCpp + Qwen3.5）に質問し、回答をVOICEVOXで読み上げます。
インターネット接続不要で、プライベートな質問にも対応できます。

回答が150文字を超える場合、読み上げは**AIが生成した140文字の要約**になります。
Discordには全文と要約の両方が表示されます。

| オプション | 必須 | 説明 |
|-----------|:----:|------|
| question | o | AIへの質問 |
| speaker | - | 話者の選択（デフォルト: ずんだもん） |
| channelid | - | ボイスチャンネルのID |

**例:**
```
/vvq question:量子コンピュータについて教えて
```

---

## `/q35` - Qwen3.5 テキスト回答

ローカルAI（KoboldCpp + Qwen3.5）にテキストのみで質問します。音声読み上げなし。
ボイスチャンネルへの参加が不要なので手軽に使えます。

| オプション | 必須 | 説明 |
|-----------|:----:|------|
| question | o | AIへの質問 |

**例:**
```
/q35 question:Pythonでリスト内包表記の書き方
```

---

## `/lvv` - ボイスチャンネルから退出

ボットをボイスチャンネルから退出させます。
10分間操作がない場合は自動的に退出します。

**例:**
```
/lvv
```

---

## 基本的な使い方

1. ボイスチャンネルに参加する
2. テキストチャンネルでスラッシュコマンドを入力
3. ボットが自動でVCに参加し、音声を再生
4. 使い終わったら `/lvv` で退出（10分放置で自動退出）

---

## 注意事項

- コマンドは**1ユーザーにつき1つずつ**順番に処理されます（並行実行不可）
- VCに未参加の場合は `channelid` でチャンネルIDを指定してください
- `/vvq`、`/q35` はローカルAIサーバー（KoboldCpp）が起動している必要があります
- `/vvai` の `search:True` でGoogle検索による最新情報の補完が可能です
- 音声ファイルは一時的に生成され、再生後に自動削除されます

---

## 話者について

VOICEVOXには多数の話者が用意されています。`speaker` オプションから選択できます。

```
話者名の例:
  ずんだもん (ノーマル)
  四国めたん (ノーマル)
  春日部つむぎ (ノーマル)
  雨晴はう (ノーマル)
  波音リツ (ノーマル)
```

選択肢にない話者は `/vv` の `speaker` で「カスタム」→ `custom_speaker` に話者名を入力。
